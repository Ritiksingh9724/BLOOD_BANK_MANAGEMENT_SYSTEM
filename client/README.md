# Blood Bank Management System

## Overview

Blood Bank Management System is a MERN Stack web application that helps hospitals request blood units and allows administrators to manage blood requests efficiently.

## Features

* User Authentication
* Hospital Registration
* Blood Request Management
* Request Approval/Rejection
* Razorpay Payment Integration
* Email Notifications
* Dashboard Analytics
* Responsive Design

## Tech Stack

### Frontend

* React.js
* Bootstrap
* Axios
* React Hot Toast

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

### Payment Gateway

* Razorpay

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/blood-bank-management-system.git
```

### Install Frontend

```bash
cd client
npm install
npm start
```

### Install Backend

```bash
cd server
npm install
npm run server
```

## Environment Variables

Create a .env file inside server folder:

```env
MONGO_URL=
JWT_SECRET=
EMAIL=
EMAIL_PASSWORD=
RAZORPAY_KEY_ID=
RAZORPAY_SECRET=
```
